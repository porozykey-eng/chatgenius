# 3 分钟上手

ChatGenius AI 需要接入一家 AI 服务商的 API 来驱动智能回复功能。整个过程只需 4 步，通常 3 分钟内即可完成。

## 配置流程

### 第 1 步：选择服务商

根据你的需求选择合适的 AI 服务商：

| 需求场景 | 推荐服务商 | 理由 |
|---------|-----------|------|
| 性价比优先 | **DeepSeek** | 价格低廉，支付宝充值，国内直连 |
| 企业稳定性 | **通义千问** | 阿里云背书，服务稳定可靠 |
| 免费体验 | **智谱清言** | GLM-4 Flash 模型免费使用 |

::: tip 不确定选哪家？
查看完整的 [服务商选型指南](/guide/choosing-a-provider) 进行对比。
:::

### 第 2 步：获取 API Key

前往服务商控制台，注册账号并创建 API Key。详细操作参考各服务商页面或 [通用配置步骤](/guide/common-steps)。

### 第 3 步：粘贴到插件

打开 ChatGenius 设置页，选择对应服务商，将 API Key 粘贴到输入框中。

### 第 4 步：验证连接

点击「测试连接」按钮，确认返回成功即可开始使用。

::: info 配置一次即可
API Key 保存在浏览器本地（`chrome.storage.local`），关闭浏览器或重启电脑后仍然有效。如需更换服务商，在设置页重新选择即可。
:::
