# 通用配置步骤

无论你选择哪家服务商，配置流程都遵循以下 4 步。

## 第 1 步：注册账号

前往服务商官网注册账号。

**国内服务商**通常需要：
- 手机号或邮箱注册
- 完成实名认证（部分厂商要求）

**国际服务商**通常支持：
- 邮箱注册
- Google / GitHub 第三方登录

## 第 2 步：获取额度

### 免费额度

多家服务商为新用户提供免费额度：

| 服务商 | 免费方案 |
|--------|---------|
| 智谱清言 | GLM-4 Flash 模型持续免费 |
| DeepSeek | 新用户赠送测试额度 |
| 通义千问 | 新人免费 Token 额度 |
| 豆包 | 新人免费额度 |
| Google AI | Gemini Flash 持续免费 |

### 充值

如需更多用量，在服务商控制台的「费用中心」或「Billing」页面充值。国内服务商通常支持支付宝和微信支付。

## 第 3 步：创建 API Key

在服务商控制台创建 API Key：

1. 登录服务商控制台
2. 找到 **API 密钥** 或 **API Keys** 页面
3. 点击 **创建 API Key**
4. 输入名称（可选），确认创建
5. **立即复制并保存** — 部分服务商仅显示一次

::: warning 安全提醒
- API Key 创建后请尽快复制，部分厂商只显示一次完整内容
- 不要将 API Key 分享给他人或提交到公开代码仓库
- 可在服务商后台设置 Key 的使用范围和额度限制
:::

## 第 4 步：配置到 ChatGenius

1. 点击浏览器工具栏中的 ChatGenius 图标，进入设置页
2. 在「服务商」下拉菜单中选择你使用的服务商
3. 将 API Key 粘贴到输入框
4. 点击「保存」

::: tip 验证配置
保存后点击「测试连接」，如果返回成功提示，说明配置正确。
:::

## 配置完成

配置完成后，ChatGenius 会自动使用你选择的 AI 模型生成智能回复。API Key 保存在浏览器本地，关闭浏览器或重启电脑后仍然有效。

如需更换服务商，在设置页重新选择并输入新的 API Key 即可。
