## 如何选择

1. **预算敏感** — DeepSeek 或智谱清言，成本最低
2. **追求稳定** — 通义千问（阿里云）或 OpenAI，大厂背书
3. **需要免费额度** — 智谱清言 GLM-4 Flash 完全免费
4. **想体验多家模型** — OpenRouter 聚合平台，一个 Key 切换多家

::: tip 可以随时切换
不确定选哪家？先选一家试用。如需更换，在 ChatGenius 设置页重新选择服务商并输入新的 API Key 即可。
:::
