# 豆包

字节跳动旗下大模型，性能强劲。

## 快速接入

1. 前往 [火山方舟控制台](https://console.volcengine.com/ark) 注册并完成实名认证。
2. 在火山方舟控制台开通豆包模型服务。新用户通常有免费额度。
3. 在 **API Key 管理** 页面创建新的密钥。
4. 复制 API Key，粘贴到 ChatGenius 设置页。

::: info 火山引擎
豆包运行在火山引擎平台上，需要火山引擎账号而非抖音账号。
:::

## 可用模型

| 模型 | 说明 |
|------|------|
| **豆包 Pro 256K** | 最新旗舰，超长上下文 |
| **豆包 Pro 32K** | 标准版本，推荐日常使用 |
| **豆包 Lite 32K** | 轻量版本 |

## 特点

- 字节跳动技术背景，性能强劲
- 支持 256K 超长上下文窗口
- 火山引擎平台，企业级稳定性
