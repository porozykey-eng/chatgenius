# Moonshot

月之暗面旗下大模型，擅长长文本处理。

## 快速接入

1. 前往 [Moonshot 开放平台](https://platform.moonshot.cn/) 注册账号。
2. 在左侧导航栏找到 **API 密钥管理**，创建新的密钥。
3. 复制 API Key（`sk-` 开头），粘贴到 ChatGenius 设置页。

::: tip
Moonshot API Key 以 `sk-` 开头，请确保完整复制。
:::

## 可用模型

| 模型 | 说明 |
|------|------|
| **Kimi Latest** | 最新模型，推荐日常使用 |
| **Moonshot V1 128K** | 长文本模型，适合大量文档分析 |
| **Moonshot V1 32K** | 标准版本 |

## 特点

- 擅长长文本处理，最大支持 128K 上下文
- Kimi 助手广受好评
- 接入简单，无需实名认证
