# Anthropic

Claude 系列模型的开发商，以安全性著称。

## 快速接入

1. 前往 [Anthropic Console](https://console.anthropic.com/) 注册账号。
2. 在 **API Keys** 页面创建新的密钥。
3. 复制 API Key（`sk-ant-` 开头），粘贴到 ChatGenius 设置页。

::: warning 网络要求
国内用户访问 Anthropic 需要稳定的网络环境。如遇连接问题，建议选择国内服务商。
:::

## 可用模型

| 模型 | 说明 |
|------|------|
| **Claude 3.7 Sonnet** | 最新模型，推荐日常使用 |
| **Claude 3.5 Sonnet** | 上一代高性能模型 |
| **Claude 3 Opus** | 旗舰模型 |

## 特点

- Claude 系列模型在安全性和准确性方面表现突出
- 超长上下文支持
- 对复杂指令的理解能力强

::: danger 支付方式
Anthropic 仅支持国际信用卡，不支持支付宝和微信支付。
:::
