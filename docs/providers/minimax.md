# MiniMax

MiniMax 旗下大模型，多模态能力强。

## 快速接入

1. 前往 [MiniMax 开放平台](https://www.minimaxi.com/) 注册账号。
2. 在左侧导航栏找到 **接口密钥**，创建新的密钥。
3. 复制 API Key，粘贴到 ChatGenius 设置页。

## 可用模型

| 模型 | 说明 |
|------|------|
| **ABAB 6.5** | 最新模型，推荐日常使用 |
| **ABAB 5.5** | 上一代版本 |

## 特点

- 多模态能力突出
- 语音和文本双模态支持
- ABAB 系列模型持续迭代
