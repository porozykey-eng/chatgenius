# 通义千问

阿里云旗下大模型，稳定可靠，新人有免费额度。

**费用估算**：日均 50 条消息 ≈ ¥0.8/天

## 快速接入

1. 前往 [DashScope 控制台](https://dashscope.console.aliyun.com/) 登录或注册。如已有阿里云账号可直接登录，需完成实名认证。
2. 开通 DashScope 服务。新用户通常有免费 Token 额度。
3. 在顶部菜单找到 **API-KEY 管理**，创建新的 API Key。
4. 复制 API Key（`sk-` 开头），粘贴到 ChatGenius 设置页。

::: info 实名认证
阿里云要求完成实名认证才能使用 DashScope 服务。认证通常在几分钟内完成。
:::
