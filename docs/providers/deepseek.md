# DeepSeek

深度求索 — 高性价比国产大模型，支持支付宝充值。

**费用估算**：日均 50 条消息 ≈ ¥1.2/天

## 快速接入

1. 前往 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册账号。支持手机号和邮箱注册。
2. 在「费用中心」充值或使用新用户赠送的免费额度。支持支付宝。
3. 在左侧导航栏点击 **API Keys**，创建新的 API Key。
4. 复制 API Key（`sk-` 开头），粘贴到 ChatGenius 设置页。

::: tip
DeepSeek API Key 以 `sk-` 开头，请确保完整复制。
:::
