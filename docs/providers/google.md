# Google AI

Gemini 系列模型，Google 出品。

## 快速接入

1. 使用 Google 账号登录 [Google AI Studio](https://aistudio.google.com/apikey)。
2. 点击 **Create API Key** 创建新的密钥。
3. 复制 API Key，粘贴到 ChatGenius 设置页。

::: tip 免费模型
Gemini 2.0 Flash 模型可免费使用，适合测试和日常使用。
:::

::: warning 网络要求
国内用户访问 Google AI 需要稳定的网络环境。如遇连接问题，建议选择国内服务商。
:::

## 可用模型

| 模型 | 说明 |
|------|------|
| **Gemini 2.5 Pro** | 最新旗舰模型（预览版） |
| **Gemini 2.0 Flash** | 快速模型，免费可用 |
| **Gemini 1.5 Pro** | 上一代旗舰 |

## 特点

- Google 技术背景，持续迭代
- Gemini Flash 系列免费使用
- 多模态能力强（文本、图片、视频）
- 超长上下文窗口支持
