# OpenRouter

聚合多家大模型的统一接口平台。一个 API Key 即可访问 OpenAI、Anthropic、Google 等多家模型。

## 快速接入

1. 前往 [OpenRouter](https://openrouter.ai/keys) 登录。支持 Google 或 GitHub 账号。
2. 在 **Keys** 页面创建新的密钥。
3. 复制 API Key（`sk-or-` 开头），粘贴到 ChatGenius 设置页。

::: tip 一个 Key 访问多家模型
OpenRouter 聚合了 OpenAI、Anthropic、Google、Meta 等多家厂商的模型，使用一个 API Key 即可切换调用。
:::

## 模型

OpenRouter 提供数百个模型，在 ChatGenius 中可动态加载。常用模型包括：

- GPT-4o / GPT-4o Mini（OpenAI）
- Claude 3.7 Sonnet（Anthropic）
- Gemini 2.0 Flash（Google）
- Llama 3（Meta）
- DeepSeek V3 / R1

## 特点

- 统一接口，一个 Key 访问多家模型
- 支持 Google / GitHub 快捷登录
- 模型种类丰富，可灵活切换
- 价格可能比直接使用原厂稍高

::: info 定价说明
OpenRouter 在各厂商原价基础上加收少量手续费。如果长期使用某一家模型，直接在原厂注册可能更经济。
:::
