# OpenAI

ChatGPT 背后的公司，全球最知名的大模型服务商。

## 快速接入

1. 前往 [OpenAI Platform](https://platform.openai.com/api-keys) 注册账号。支持邮箱或 Google 账号。
2. 在 **Billing** 页面添加支付方式并充值。支持国际信用卡。
3. 在 **API Keys** 页面点击 **Create new secret key**，创建新的密钥。
4. 复制 API Key（`sk-` 开头），粘贴到 ChatGenius 设置页。

::: warning 网络要求
国内用户访问 OpenAI 需要稳定的网络环境。如遇连接问题，建议选择国内服务商。
:::
