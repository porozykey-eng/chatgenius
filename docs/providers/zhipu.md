# 智谱清言

清华系大模型，提供免费模型可用。

**费用估算**：GLM-4 Flash 模型免费使用

## 快速接入

1. 前往 [智谱开放平台](https://open.bigmodel.cn/) 注册账号。支持手机号注册。
2. 新用户注册后可领取免费 Token 额度。GLM-4 Flash 模型持续免费。
3. 在顶部菜单找到 **API 密钥**，创建新的密钥。
4. 复制 API Key，粘贴到 ChatGenius 设置页。

::: warning Key 格式特殊
智谱的 API Key 为 JWT 格式（`xxxxxxxx.xxxxxxxx`），比其他厂商的 Key 更长，请确保完整复制。
:::
