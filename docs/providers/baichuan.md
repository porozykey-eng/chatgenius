# 百川大模型

百川智能旗下大模型，中文能力出色。

## 快速接入

1. 前往 [百川开放平台](https://platform.baichuan-ai.com/) 注册账号。
2. 在左侧导航栏找到 **API Keys**，创建新的密钥。
3. 复制 API Key，粘贴到 ChatGenius 设置页。

## 可用模型

| 模型 | 说明 |
|------|------|
| **Baichuan 4** | 最新旗舰模型 |
| **Baichuan 3 Turbo** | 快速版本 |

## 特点

- 中文理解和生成能力出色
- 适合中文内容创作场景
- 接入简单
