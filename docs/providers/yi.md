# 零一万物

李开复创办的 AI 公司旗下大模型。

## 快速接入

1. 前往 [零一万物开放平台](https://platform.lingyiwanwu.com/) 注册账号。
2. 在左侧导航栏找到 **API 密钥**，创建新的密钥。
3. 复制 API Key，粘贴到 ChatGenius 设置页。

## 可用模型

| 模型 | 说明 |
|------|------|
| **Yi Lightning** | 最新模型，推荐日常使用 |
| **Yi Large** | 大参数模型 |

## 特点

- Yi 系列模型性能优异
- 中英文双语能力均衡
