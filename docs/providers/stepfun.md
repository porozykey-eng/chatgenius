# 阶跃星辰

阶跃星辰旗下大模型。

## 快速接入

1. 前往 [阶跃星辰开放平台](https://platform.stepfun.com/) 注册账号。
2. 在左侧导航栏找到 **API 密钥**，创建新的密钥。
3. 复制 API Key，粘贴到 ChatGenius 设置页。

## 可用模型

| 模型 | 说明 |
|------|------|
| **Step 2 16K** | 最新模型，推荐日常使用 |
| **Step 1 8K** | 标准版本 |

## 特点

- Step 系列模型持续升级
- 接入简单，注册即可使用
