3 分钟快速配置

无论您选择哪家服务商，配置流程都是相同的

📋 通用配置步骤
  1. 注册账号
    访问服务商官网，注册并完成实名认证（国内厂商通常需要）
  2. 充值或获取免费额度
    部分服务商提供新人免费额度，付费用户可按需充值
  3. 创建 API Key
    在控制台找到「API 密钥」或「API Keys」页面，创建一个新的密钥
  4. 复制密钥到插件
    复制生成的 API Key，粘贴到 ChatGenius 设置页的对应输入框中
💡 小提示
API Key 只需配置一次，之后会保存在您的浏览器本地。如需更换服务商，可随时在设置页修改。

常见问题

配置过程中可能遇到的问题及解决方案

API Key 在哪里找？
每家服务商的 API Key 位置略有不同，但通常都在「控制台」→「API 密钥」或「API Keys」菜单下。

通用查找路径：
  1. 登录服务商官网
  2. 进入「控制台」或「管理后台」
  3. 找到「API」或「密钥管理」相关菜单
  4. 点击「创建 API Key」
测试连接失败怎么办？
常见原因及解决方法：
  • Key 无效（401/403）：检查是否复制完整，注意不要多复制空格
  • 余额不足（429）：需要先在服务商后台充值
  • 网络问题：国内用户建议选择国内服务商，访问更稳定
  • Key 权限不足：部分服务商需要单独开通 API 权限
API Key 安全吗？会泄露吗？
您的 API Key 仅保存在浏览器本地（chrome.storage.local），不会上传到我们的服务器。

安全建议：
  • 不要分享您的 API Key 给他人
  • 定期检查服务商后台的用量，发现异常及时更换 Key
  • 部分服务商支持设置 Key 的使用范围和额度限制
可以使用多家服务商吗？
目前插件支持配置一家服务商。如果您想切换到其他服务商，只需在设置页重新选择并输入新的 API Key 即可。
如何查看我的用量和费用？
用量和费用由您选择的服务商管理。请登录服务商后台查看详细的调用记录和账单。

插件本身不会产生额外费用，您只需支付服务商的 API 调用费用。

🟠 DeepSeek 配置指南

深度求索 - 高性价比国产大模型，支持支付宝充值

1
注册 DeepSeek 账号
访问 DeepSeek 开放平台，使用手机号或邮箱注册账号
前往注册 →
2
充值或使用免费额度
新用户通常有免费额度可供测试。如需更多用量，可在「费用中心」使用支付宝充值
3
创建 API Key
在左侧菜单找到「API Keys」，点击「创建 API Key」，输入名称后确认创建
4
复制并粘贴到插件
点击复制按钮获取 Key（格式通常为 sk- 开头），返回 ChatGenius 设置页粘贴即可
💡 提示
DeepSeek 的 API Key 以 sk- 开头，请确保复制完整。
