第一步：安装插件

1
打开 Chrome 应用商店
访问 Chrome Web Store，搜索 "ChatGenius"
2
安装插件
点击 "Add to Chrome" → "Add extension" 确认安装
3
确认安装成功
浏览器右上角出现 ChatGenius 图标（💬），点击图标打开插件

安装完成后
安装后，浏览器右上角会出现 ChatGenius 图标，点击即可打开插件面板。

第二步：配置大模型

插件安装完成后，需要配置一个 AI 服务商的 API Key 来驱动对话功能。

1
注册 AI 服务商账号
选择一个服务商（推荐 DeepSeek 或通义千问），在官网注册并完成实名认证。
2
创建并复制 API Key
登录控制台，进入「API Key 管理」创建 Key，创建后立即复制保存。
3
在 ChatGenius 中粘贴
打开设置页 → 选择服务商 → 粘贴 Key → 点击「测试连接」。

API Key 保存在浏览器本地存储中，仅存在于您的设备上，不会上传到任何第三方。可随时在设置中删除或更换。
DeepSeek 以 sk- 开头；智谱 AI 含点号；OpenAI 以 sk-proj- 开头；Anthropic 以 sk-ant- 开头。插件会自动校验格式。
ChatGenius 是 Chrome 扩展，支持 Android Chrome。iOS Safari 不支持扩展。手机端配置方式与电脑相同。
中国大陆用户强烈建议选择国内服务商（DeepSeek、通义千问等），注册简单、无需信用卡、直连速度快。国际服务商需要海外网络和国际信用卡。
常见原因：Key 复制不完整、余额不足、网络问题、Key 已过期。解决：重新复制 Key → 检查余额 → 确认网络 → 重新创建 Key。
