第一步 · 安装插件

访问 Chrome 应用商店，搜索 ChatGenius 并安装。安装后浏览器右上角会出现插件图标。

前往 Chrome 商店安装

第二步 · 配置 API

ChatGenius 需要连接一个大语言模型服务商。选择下方推荐的服务商，点击预设按钮自动填充配置，然后填入你的 API Key 即可。

什么是 API Key？
API Key 是访问大模型服务的「通行证」。您在服务商官网注册账号后，在控制台创建一个 Key，把它粘贴到下方即可。Key 仅保存在本地浏览器，不会上传服务器。
API 配置
选择服务商预设，或自定义配置
API URL 服务商接口地址
API Key 在服务商控制台创建
模型名称 对话使用的模型 ID
配置后如何使用？
复制上方的 URL、Key、模型名称，打开 ChatGenius 插件设置页，粘贴到对应输入框并保存即可。或点击「复制全部」一次性复制所有配置。

第三步 · 开始使用

配置完成后，打开 WhatsApp、Messenger 或 Facebook 网页版，点击浏览器右上角的 ChatGenius 图标，即可开始 AI 对话。
