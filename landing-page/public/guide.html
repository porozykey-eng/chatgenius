快速配置

只需 3 步，即可让 ChatGenius 连接到大语言模型。整个过程不超过 2 分钟。

什么是 API Key？

API Key 是您访问大模型服务的「通行证」。您可以把它理解为一个专属密码——有了它，ChatGenius 就能代替您向 AI 服务商发送请求并获取回答。

费用说明：API 调用按实际使用量计费（通常按 token 计费）。日常聊天每 1000 条消息约 1-5 元人民币，远低于人工成本。大多数服务商提供免费试用额度。

安全性：您的 API Key 仅保存在浏览器本地存储中，不会上传到 ChatGenius 服务器。您可以随时在插件设置中删除或更换。

三步完成配置

1
注册 AI 服务商账号
选择一个 AI 服务商（推荐从 DeepSeek 或通义千问开始），在其官网注册账号并完成实名认证。大多数服务商支持手机号快速注册。
2
创建并复制 API Key
登录服务商控制台，找到「API Key 管理」页面，点击「创建 Key」。创建成功后立即复制保存——部分服务商只在创建时显示一次 Key。
3
在插件中粘贴 Key
打开 ChatGenius 设置页，选择对应的服务商，将复制的 API Key 粘贴到输入框中，点击「测试连接」验证成功即可。

使用过程中遇到问题？这里可能有您需要的答案。
