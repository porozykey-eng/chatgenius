什么是 API Key？
API Key 是访问大模型服务的「通行证」。你在服务商官网注册账号后，在控制台创建一个 Key。Key 仅保存在本地浏览器，不会上传服务器。
如何找到模型名称？
登录服务商控制台后，在「模型列表」或「文档」中查看当前可用的模型 ID。模型会不定期更新，请以服务商官网为准。

第三步 · 配置大模型

准备好 API Key 后，你需要填写三项信息：API URL、API Key、模型名称。

前往 大模型接入 章节使用配置器，可在线测试连接是否成功，再复制到插件设置页。

第四步 · 开始使用

配置完成后，打开 WhatsApp、Messenger 或 Facebook 网页版，点击浏览器右上角的 ChatGenius 图标，即可开始 AI 对话。

中国大陆用户建议优先选择国内服务商（DeepSeek、通义千问、智谱等）：注册简单、无需信用卡、国内直连速度快、价格低、中文能力强。国际服务商模型更强，但需海外网络和信用卡。
常见原因：1) API Key 复制不完整或含多余空格；2) 账户余额不足；3) 网络问题（国际服务商需代理）；4) 模型名称写错或已下架；5) API URL 格式不正确。解决：重新复制 Key → 检查余额 → 确认网络 → 核对模型名称 → 检查 URL 是否以 /v1 结尾。
