快速配置指南

无论您选择哪家 AI 服务商，配置流程都是相同的。只需 4 步，即可在 WhatsApp 等平台使用 AI 自动回复。

1
注册服务商账号
访问服务商官网，注册并完成实名认证（国内厂商通常需要手机号验证）。详见左侧各服务商配置页。
2
获取 API 额度
部分服务商提供新人免费额度，可先用于测试。付费用户可按需充值。
3
创建 API Key
在服务商控制台找到「API 密钥」或「API Keys」页面，点击创建一个新的密钥并复制。
4
粘贴到 ChatGenius
打开插件设置页，选择对应的服务商，将 API Key 粘贴到输入框，点击「测试连接」验证成功即可。
✓
只需配置一次
API Key 保存在浏览器本地，不会上传到我们的服务器。如需更换服务商，在设置页重新选择即可。

常见问题

配置过程中可能遇到的问题及解决方案。

API Key 在哪里找？
每家服务商的 API Key 位置略有不同，但通常都在「控制台」→「API 密钥」或「API Keys」菜单下。

通用查找路径：
① 登录服务商官网
② 进入「控制台」或「管理后台」
③ 找到「API」或「密钥管理」相关菜单
④ 点击「创建 API Key」
测试连接失败怎么办？
常见原因及解决方法：
• Key 无效（401/403） — 检查是否复制完整，注意不要多复制空格
• 余额不足（429） — 需要先在服务商后台充值
• 网络问题 — 国内用户建议选择国内服务商
• Key 权限不足 — 部分服务商需要单独开通 API 权限
API Key 安全吗？会泄露吗？
您的 API Key 仅保存在浏览器本地，不会上传到我们的服务器。

安全建议：
• 不要分享您的 API Key 给他人
• 定期检查服务商后台的用量，发现异常及时更换 Key
• 部分服务商支持设置 Key 的使用范围和额度限制
可以使用多家服务商吗？
目前插件支持配置一家服务商。如需切换，在设置页重新选择并输入新的 API Key 即可。
如何查看用量和费用？
用量和费用由您选择的服务商管理。请登录服务商后台查看详细的调用记录和账单。插件本身不会产生额外费用。

注册账号

前往 DeepSeek →
2
获取额度

3
创建 API Key
